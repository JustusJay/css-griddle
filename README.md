# Slide Puzzle

A modern, clean implementation of the classic 15-puzzle game.

## Framework
Vanilla JavaScript (Static HTML) with Tailwind CSS.

## Features
- Classic 15-puzzle mechanics (4x4 grid)
- Guaranteed solvable shuffling algorithm
- Move counter and timer
- Responsive design (Mobile & Desktop)
- Clean, modern UI using Tailwind CSS
- Win state detection

## Development
This is a static HTML application. You can serve it using any static file server.

```bash
# Example using python
python3 -m http.server 8080
```

## How to Play
1. Click on any tile adjacent to the empty space to move it.
2. Order the numbers from 1 to 15, left to right, top to bottom.
3. The empty space should end up in the bottom-right corner.
